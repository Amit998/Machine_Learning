## #Activation

## Identity= f(x)=1
## Binary Step= 0 / 1
## Logistic or Sigmoid= 1/1+e^-x
## ReLU(Rectified Linear Unit)=  f(x){ 0 for x < 0 or x for x =>0 }
## Leaky Relu=  f(x){ 0.01x for x < 0 or x for x =>0 }
## ArcTan= tan^-1(x)
## tanh= (2 / 2 + e^-2x)-1
## softmax= e ^zj/ {k or k=1^ezk}  for j = 1,.....k
